Clone the project

```bash
  git clone https://link-to-project
```

Go to the project directory

```bash
  cd my-project
```

## Run Locally

Install dependencies

```bash
  yarn
```

Start the server

```bash
  yarn dev
```

## Run Production

Install dependencies

```bash
  yarn
```

Build

```bash
  yarn build
```

Start the server

```bash
  yarn start
```
